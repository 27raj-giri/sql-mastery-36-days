# SQL Mastery in 36 Days

This repository documents my structured journey of mastering SQL through consistent daily problem solving.

The goal is not just to learn SQL syntax, but to build strong analytical thinking by applying SQL across progressively complex scenarios.

Over 36 days, this repository captures:

- Understanding problems
- Writing solutions
- Reflecting on learnings
- Mapping SQL to real-world use cases

---

## Learning Areas Covered

Throughout the journey, the following key SQL capabilities are developed:

- Data Filtering
- CRUD Operations
- Aggregations
- Joins
- Subqueries
- Functions
- Window Functions
- Case Logic
- Query Optimization
- Business Analysis
- Interview-Oriented SQL

---

## Approach

The focus of this journey is:

Solve → Understand → Document → Apply

Instead of rushing through topics, each day emphasizes:

- Concept clarity
- Practical execution
- Business thinking

## Objective

To build a strong SQL foundation for:

- Data Analytics
